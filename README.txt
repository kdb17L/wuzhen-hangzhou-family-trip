家庭旅游手机网页使用说明

1. 怎样打开
   双击 index.html，即可用电脑或手机浏览器离线打开。微信内置浏览器也可查看；地图、官方渠道、小红书和大众点评按钮需要联网。

2. 怎样替换门票图片
   在 index.html 同级位置新建 assets/tickets 文件夹，放入凭证图片；然后在 index.html 的 tripDays 数据中找到对应门票，把 ticketImage: "" 改为例如：ticketImage: "assets/tickets/lingyin.jpg"。
   请勿放置身份证号、完整手机号或动态二维码。

3. 怎样修改行程
   用记事本或其他文本编辑器打开 index.html，搜索“修改日期、每日概览、时间轴和门票数据的位置”，在 tripDays 数组内修改。联系人请搜索“修改联系人信息的位置”。导航地点请修改各节点的 nav 字段。

4. 仍待人工确认
   Day1 到店、入住、早餐和住客权益；Day5 缺失项目、油纸伞项目及午餐；部分餐厅全名与门店；乌镇连续往返杭州的车程和出发时间；德寿宫放票日期矛盾；雷峰塔转场；所有预约规则、票价、开放时间、酒店权益和返程订单。

未见实际订单或凭证的项目，网页统一显示为待预约、待购票、待添加凭证或请再次确认。
